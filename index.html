无需设计：快速搭建漂亮页面；同时支持电脑和手机！

TealUI 提供了 200 多个常用 UI 组件，无论是整库使用还是单独使用个别组件，复制示例代码即可见效，非常方便。

更懂国情：适合中文排版

TealUI 提供了城市选择、拼音汉字转换、身份证验证等业务组件。同时 TealUI 也兼顾国内浏览器占比，支持更多国产浏览器。

一个框架、多种设备

只需写一份代码，即可同时适应 PC、平板、手机等不同尺寸的设备。所有组件考虑移动优先，为手机平台更省流量。

轻量级、高性能

所有组件功能专一、依赖性小、代码轻量。即使不使用整个 UI 库，每个组件也可以独立使用。
