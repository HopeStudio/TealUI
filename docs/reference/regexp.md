常用正则
==========================================================

<dt>网络</dt>

<dd>Email: `/\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*/`</dd>

<dd>网址: `/^[a-zA-z]+:///`</dd>

<dd>HTML: `/<(\s*?)[^>]*>.*?|<.*? />/`</dd>

<dd>首尾空白字符: `/^\s*|\s*$/g`</dd>

<dt>中文</dt>

<dd>中文: `/\u4e00-\u9fa5/`</dd>

<dd>身份证: `/\d{15}|\d{18}/`</dd>
