TealUI 目录结构规范

```
TealUI/
    ├── assets/         开发系统所需的资源文件
    ├── demos/          组件完整 DEMO 展示
    ├── docs/           文档和教程
    ├── src/            组件源码
    │   ├── 3rdlib/     第三方独立框架
    │   ├── control/    PC 端 UI 组件
    │   ├── node/       NodeJs 专用组件
    │   ├── touch/      移动端 UI 组件
    │   ├── typography/ CSS 框架
    │   ├── utility/    JS 工具函数
    │   ├── wechat/     微信专用组件
    │   └── widget/     业务组件
    └── tools/          相关工具
```
目标，所有模块可以按需使用。
