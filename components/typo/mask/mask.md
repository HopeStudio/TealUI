# 遮罩
遮罩（蒙版）主要用于覆盖用户点击区域，防止用户的误操作。

## 基本用法
```html demo
我是普通文本
<div class="x-mask">
    <div class="x-mask-body"><i class="x-icon x-spin">&#1161;</i> 加载中...</div>
</div>
```
