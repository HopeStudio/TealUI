# 导航菜单
简单的导航项

## 基本用法
```html demo
<ul class="x-nav">
    <li class="x-nav-active"><a href="###">首页</a></li>
    <li><a href="###">详情</a></li>
    <li><a href="###">关于我们</a></li>
</ul>
```

## 水平布局
```html demo
请选择：<ul class="x-nav x-nav-inline">
    <li class="x-nav-active"><a href="###">首页</a></li>
    <li><a href="###">详情</a></li>
    <li><a href="###">关于我们</a></li>
</ul>
```

## 导航和徽章
```html demo
<ul class="x-nav x-nav-inline">
    <li class="x-nav-active"><a href="###">首页<span class="x-badge">2</span></a></li>
    <li><a href="###">详情</a></li>
    <li><a href="###">关于我们<span class="x-badge">1</span></a></li>
</ul>
```
