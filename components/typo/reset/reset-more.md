# 样式重置扩展
更多的样式重置。

1. 隐藏 `tabindex = -1` 的边框。
2. 令 IE6-8 支持 HTML5 标签。
3. 修复更多表单标签的样式。

## `<fieldset>`
```html demo
<fieldset>
    <legend>标签</legend>
    文字
</fieldset>
```

## `<abbr>`
```html demo
<abbr title="我是说明">abbr</abbr>
```
