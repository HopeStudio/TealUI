    ☍键盘按下事件web/keyPress

    绑定常用键盘按键。

    ❐import keyPress from "web/keyPress";

keyPress(elem, {
    up() {
        console.log("上");
    },
    enter() {
        console.log("回车");
    }
});

    绑定指定元素的键盘按键事件。

    元素。

    		 —
    options* KeyPressOptions

    由键值和要绑定的事件函数组成的键值对。
