# 浏览器平台检测
让浏览器强制打开指定网页。

        document.getElementById('ua').innerHTML = navigator.userAgent;

##### 获取浏览器名
```
navigator.browser
```

##### 获取浏览器主版本
```
navigator.version
```

##### 获取浏览器引擎
```
navigator.engine
```

##### 获取操作系统
```
navigator.platform
```
