    ☍API

    函数 描述
    ⅏ ⮞ getPinYin(value)(value:string):string[][]

    获取中文的拼音。

    参数 类型 描述 默认值
    value* string

    中文字符串。

    		 —

    返回值

    类型：string[][]

    返回每个字符发音组成的数组。如果不是中文或无法识别则该字符对应的位置是空数组。

    获取中文的拼音。
    ⅏ ⮞ getPinYinOfChar(value)(value:string):string[]

    获取单个中文字符的拼音。

    参数 类型 描述 默认值
    value* string

    单个中文字符。

    		 —

    返回值

    类型：string[]

    返回字符的所有发音。如果不是中文或无法识别则返回空数组。

    获取单个中文字符的拼音。
