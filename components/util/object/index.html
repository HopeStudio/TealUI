    ⅏ ⮞ assignIf(target, source)<T, R>(target:T, source:R):T & R

    复制源对象的所有可枚举属性到目标对象，如果目标对象中对应的属性值不是 undefined 则跳过。

    泛型参数 约束类型 默认类型
    T — —
    R — —
    参数 类型 描述 默认值
    target* T

    目标对象。

    		 —
    source* R

    源对象。源对象可以是 null 或 undefined。

    		 —

    返回值

    类型：T & R

    返回目标对象。

    示例

    ❐assignIf({a: 1}, {a: 2, b: 2}) // {a: 1, b: 2}

    复制源对象的所有可枚举属性到目标对象，如果目标对象中对应的属性值不是 undefined 则跳过。
