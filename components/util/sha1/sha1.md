# SHA-1 加密
纯 JavaScript 实现 SHA-1 加密算法。

```html demo hide doc
<input type="text" id="input" style="width: 26em;" placeholder="要加密的内容" />
<button onclick="input.value = sha1(input.value)">SHA-1 加密</button>
```
