    ☍API

    函数 描述
    ⅏ ⮞ sha1(value)(value:string):string

    计算字符串的 SHA-1 值。

    参数 类型 描述 默认值
    value* string —

    返回值

    类型：string

    返回 40 位的全小写 SHA-1 值。

    示例

    ❐sha1("abc") // "a9993e364706816aba3e25717850c26c9cd0d89d"

    计算字符串的 SHA-1 值。
