网页的基本布局方法为：将网页先分成 N 行，每行分成 N 列。栅格法即将每行先分成 12 列，然后将任意 N 列合并，最终实现左窄右宽、左中右平分等常见分列布局效果。
栅格布局会根据屏幕自动调整大小。调整浏览器窗口大小可以查看效果。

栅格布局法

使用 .x-grid 表示一行，使用 .x-col 表示一列，列的宽度分别使用.x-col-*(其中 * 是 0-12 的整数) 标识。只要一行内的所有列宽度后缀加起来等于 12，即可占满一行。

响应式布局

通过不同的前缀，可以为不同分辨率指定不同的分列方式：

  • .x-col-*: 小屏（默认）按此宽度布局。
  • .x-col-large-*: 大屏按此宽度布局。
