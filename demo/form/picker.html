Picker 是一个文本框和按钮的组合框组件基类。选择框(combobox)、日期选择器(datepicker) 等都是继承于此类实现的。

HTML 结构

Picker 标准结构如上。Picker 会在 init 中会对不满足上述结构的 HTML 重新渲染。

1. 只需写 <input> 即可渲染一个 Picker

2. 只需写 <a> 即可渲染一个 Picker

3. 仅动态渲染一个按钮

4. 绑定下拉菜单

Picker 会自动将节点紧跟的 x-dropdown 作为下拉菜单。 最常见的做法是使用 DropDownMenu 作为下拉菜单。

Picker 使用 IDropDownOwner 创建和管理下拉菜单。

常用 API

要控制文本框， 参考 IInput 组件。(Picker 实现了IInput 接口)
要控制下拉菜单， 参考 IDropDownOwner 组件。(Picker 实现了 IDropDownOwner 接口)
