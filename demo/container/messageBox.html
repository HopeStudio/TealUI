经典消息框，可代替 alert。如果需要一个空的对话框，请使用 对话框（dialog） 组件。

基本用法

警告框(模拟 alert)

            $.alert('内容', '我是标题');

确认框(模拟 confirm)

            $.confirm('内容', '我是标题', function(){alert("点击了确定")});

输入框

        $.prompt('请输入文字', '我是标题', function(content){alert(content)});

确认框

        $.tip('发送成功。
{time} 秒后刷新页面...', '我是标题', 3000, function(){location.reload()});

标准用法

        $.messageBox()
            .title('我是标题')
            .content('内容')
            .icon("info")
            .buttons({
                '忽略': true,
                '重试': function () {
                    this.close();
                    alert("点击了重试");
                },
                '取消': false
            });
其它 API　请参考信息框的基类： 对话框（dialog） 组件。
