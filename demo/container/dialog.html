创建最简单的对话框

如果需要一个含有确定/取消按钮或者含有一个图标的对话框，请使用 信息框（messageBox） 组件。

基于已有的 HTML 创建对话框

常用 API

自定义遮罩层（模式对话框）

不显示遮罩层

        $.dialog().mask(false);

显示透明度为 0.5 的遮罩层

        $.dialog().mask(0.5);

关闭按钮

不显示关闭按钮

        $.dialog().closeButton(false);

手动关闭对话框

        $.dialog().close();

设置用户关闭对话框后的回调

        $.dialog().on("close", function(){
            alert("对话框已经关闭了");
        });

设置用户关闭对话框后的回调，并阻止关闭事件

        $.dialog().on("closing", function(){
            alert("对话框即将关闭");
            return false; // return false 阻止事件，这样将关闭不了对话框。
        });
    
        $.dialog({closeButton: false}); // 不显示关闭按钮。
        $.dialog().close();     // 手动关闭对话框。        
        $.dialog().on("close", function(){alert("对话框已经关闭了");});    // 设置关闭后的回调。        
        $.dialog().on("close", function(){alert("对话框已经关闭了");});    // 设置关闭后的回调。
