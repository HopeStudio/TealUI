TealUI 4.0 全新升级 ！
===============================================
TealUI 是专业的大型前端开源代码库。

TealUI = CSS 样式库 + UI 组件库 + JS 工具函数库 + 前端项目规范 + 完整网站模板 + ...

* 无需设计：快速搭建漂亮页面；同时支持电脑和手机！
* 300+ 个组件：一键复制使用，快速上手。
* 更懂国情：适合中文排版，从不乱码，提供城市组件，为国内需求定制。
* 轻量、完整、专业、高性能：完全开源和免费。

开源协议
-----------------------------------------------
TealUI 完全免费和开源，可用于商业项目。
但不得基于本库开发新的组件库库并再发布或销售。
