添加到收藏夹 (源码: utility/misc/browser.js)

API 描述 示例
addFavorite 弹出添加到收藏夹对话框
添加当前网页到收藏夹
addFavorite();
添加指定网页到收藏夹
addFavorite("TealUI", "http://teal.github.io/TealUI");

注意

最新浏览器由于安全限制，不允许使用此功能。这时，函数会提示用户手动操作。

设为主页 (源码: utility/misc/browser.js)

API 描述 示例
setHomePage 弹出设为主页对话框
设置当前网页为主页
setHomePage()
设置指定网页为主页
setHomePage("http://www.TealUI.com/")

注意

最新浏览器由于安全限制，不允许使用此功能。这时，函数会提示用户手动操作。
