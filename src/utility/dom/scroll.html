API 描述 示例

滚动显示事件 (源码: utility/dom/scrollShow.js)

API 描述 示例
dom.isScrollIntoView 判断是否在视图内 
$(elem).isScrollIntoView();
dom.scrollShow 元素被滚动显示后回调
每次回调
$(elem).scrollShow(function(){trace('滚动到了')});
首次回调
$(elem).scrollShow(function(){trace('滚动到了')}, 'once');
