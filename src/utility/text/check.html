常用 API (源码: utility/text/check.js)

API 描述 示例
Check.isEmail 判断字符串是否为合法邮箱地址 
Check.isEmail("work@xuld.net")
Check.isInteger 判断字符串是否为整数 
Check.isInteger("12")
Check.isNumber 判断字符串是否为数字 
Check.isNumber("12.3")
Check.isDate 判断是否为日期 
Check.isDate("2012/1/1")
Check.isLetter 判断是否为字母 
Check.isLetter("a")
Check.isDight 判断是否为数字 
Check.isDight("a")
Check.isLetterOrDight 判断是否为字母或数字 
Check.isLetterOrDight("a")
Check.isIndentifier 判断是否为 JavaScript 标识符 
Check.isIndentifier("a")
Check.isUrl 判断是否为网址 
Check.isUrl("http://teal.github.io/TealUI")
Check.isIP 判断是否为 IP 地址 
Check.isIP("127.0.0.1")
Check.isPhone 判断是否为手机号 
Check.isPhone("121100000000")
Check.isChinese 判断是否为中文 
Check.isChinese("中文")
Check.isPostCode 判断是否为邮编 
Check.isPostCode("310000")
Check.isId 判断是否为身份证号 
Check.isId("121100000000")
Check.isQQ 判断是否为 QQ 号 
Check.isQQ("10000")

输入身份证号码:

API 描述 示例
checkPassword 检查密码复杂度 
checkPassword("123456") //如果复杂度 < 0，认为是简单， = 0，认为是普通， > 0，认为是复杂， > 2，认为是安全。
