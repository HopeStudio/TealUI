API 描述 示例
Path.resolve 切换目录 
Path.resolve("a/b", "../", "c")
Path.normalize 规划化路径 
Path.normalize("a/b/../c/d/e")
Path.join 组合路径 
Path.join("a", "d")
Path.relative 获取相对路径 
Path.relative("a/b", "a/c")
Path.relative 获取相对路径 
Path.relative("a/b", "a/c")
Path.dirname 获取文件夹名 
Path.dirname("a/b/c")
Path.basename 获取文件名 
Path.basename("a/b/c")
Path.extname 获取扩展名 
Path.extname("a.js")
