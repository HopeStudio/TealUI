获取 URL 查询参数（如 ?a=b&c=d） (源码: utility/text/getQuery.js)

API 描述 示例
getQuery 解析查询字符串 
getQuery("a"); // 获取不到则返回 null。

追加查询字符串参数 (源码: utility/text/setQuery.js)

API 描述 示例
appendQuery 在指定的地址追加查询字符串参数 
appendQuery("a.index", "a", "b");

完整解析查询字符串 API (源码: utility/text/queryString.js)

API 描述 示例
QueryString.parse 解析查询字符串 
QueryString.parse(location.search);
QueryString.stringify 格式化查询字符串 
QueryString.stringify({ a: "2", c: "4" });

完整解析 URL 地址 API (源码: utility/text/uri.js)

API 描述 示例
Uri.parse 解析一个 URL 地址相关信息 
Uri.parse("http://www.domain.com:55/file.ext?a=3#k")
