常用 API

Object.assign: 复制成员
Object.assign({}, {a: 1}); // 复制 {a: 1} 的成员到 {}，并返回 {}
Object.each: 遍历对象或数组
        Object.each([1, 2, 3], function(item, index){
            console.log(index, "=>", item);
        });
        Object.each({a:1, b: 2}, function(item, key){
            console.log(key, "=>", item);
        });

不常用 API

API 描述 示例
Object.type 获取类型 
Object.type(1);
Object.isObject 判断是否是对象 
Object.isObject({});
Object.type: 获取类型
Object.type(1);
Object.isObject: 判断是否是对象
Object.isObject({});
Object.isFunction: 判断是否是函数
Object.isFunction(function(){});
Object.assignIf: 不覆盖复制成员
Object.type(1);
Object.clone: 对象深拷贝
Object.clone(['a', 'b']);
Object.count: 获取对象的属性数
Object.count({a:1});
Object.keys: 获取对象的键
Object.keys({a:1});
Object.values: 获取对象的值
Object.values({a:1});
Object.keyOf: 根据值获取键
Object.keyOf({a:1}, 1);
Object.insertBefore: 在指定位置插入键
Object.insertBefore({a:1}, 'a', 'b', 2); // {b:2, a: 1}
Object.map: 对象更新为新对象
Object.map(['a', 'b'], function(a){return a + a;});
Object.filter: 对象更新为新对象
Object.filter(['a', 'b'], function(a){return 3;});
Object.every: 判断对象是否都满足要求
Object.every(['a', 'b'], function(a){return true;});
Object.some: 判断对象是否有满足要求
Object.some(['a', 'b'], function(a){return true;});
Object.subset: 获取对象指定键的子集
Object.subset({a:1}, 'a'); // 1
Object.pick: 获取参数中第一个不为 undefined 的值
Object.pick(undefined, 1); // 1
