# HTML5 兼容补丁

@author xuld@vip.qq.com
@description 为低版本浏览器（B、C 类，包括 IE8 以及老版本 FireFox, Safari 和 Chrome）添加标准的 HTML5 常用 API。建议先引入本组件，以确保本库能兼容所有低版本浏览器。
