常用 API

API 描述 示例
arr.remove 删除当前数组中指定的元素 
[1, 2].remove(2);
arr.unique 删除当前数组的重复元素 
[1, 2].unique();

不常用 API

API 描述 示例
Array.from 将一个类数组对象转为原生数组 
Array.from({length: 1, "0": "value"});
arr.every 遍历一个数组并调用指定的函数，判断是否都满足条件 
[1, 2].every(function(){return true});
arr.some 遍历一个数组并调用指定的函数，判断是否部分满足条件 
[1, 2].some(function(){return true});
arr.map 遍历一个类数组对象并调用指定的函数，返回每次调用的返回值数组 
[1, 2].map(function(){return true});
arr.concat 合并两个数组 
[1, 2].concat([4, 4]);
arr.insert 将指定的值插入到当前数组的指定位置 
[1, 2].insert(0, 4);
arr.clean 删除数组中的空值 
[1, 2].clean()
arr.min 获取数组中的最小值 
[1, 2].min()
arr.max 获取数组中的最大值 
[1, 2].max()
arr.sum 获取数组中的和 
[1, 2].sum()
arr.avg 获取数组中的算术平均值 
[1, 2].avg()
arr.associate 将数组中的值和指定的键表组合为对象 
[1, 2].associate(["q", "w"])
arr.clear 清空数组 
[1, 2].clear()
arr.pick 获取数组中第一个不为空的元素 
[1, 2].pick()
arr.random 返回数组的随机位置的值 
[1, 2].random()
arr.shuffle 随机打乱数组的内容 
[1, 2].shuffle()
arr.flatten 将多维数组合并为一维数组 
[1, 2].flatten()
arr.checkRepeat 检查数组中是否存在重复值 
[1, 2].checkRepeat()
arr.sub 将当前数组中减去另一个数组的元素，返回剩下的元素数组 
[1, 2].sub([1])
arr.pushIf 如果不存在则添加 
[1, 2].pushIf(1)
