常用 API (源码: utility/lang/number.js)

API 描述 示例
num.format 替换字符串内全部子字符串 
1..format("000.00");
元字符 意义 实例
0 空位补0 000:012, 0.00: 2.00
+ 正负号 +: +
. 小数点 .: .
, 每三位追加逗号 ,: 0

不常用 API (源码: utility/lang/numberEx.js)

API 描述 示例
Number.isNumber 判断是否是数字 
Number.isNumber(7);
Number.limit 确保指定数字在指定区间内 
Number.limit(1, 2, 6);
Number.random 返回在指定范围的随机值 
Number.random(2, 6);
