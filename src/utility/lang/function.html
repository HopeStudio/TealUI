标准 API (源码: utility/lang/html5.js)

以下 API 由于使用频率高已被移入 HTML5 兼容补丁 组件。

API 描述 示例
(ES5)fn.bind 绑定函数 this 
(function(){ console.log(this) }).bind(1)();

常用 API (源码: utility/lang/function.js)

API 描述 示例
Function.empty 空函数 
Function.empty();
Function.from 返回一个值，这个值是当前的参数 
Function.from(false)();

不常用 API (源码: utility/lang/functionEx.js)

API 描述 示例
Function.isFunction 判断是否是函数 
Function.isFunction(function(){});
Function.concat 返回一个值，这个值是当前的参数 
Function.concat(function(){}, function(){})();
Function.tryThese 尝试运行返回正确的才内容 
Function.tryThese(function(){});
Function.interval 每隔时间执行一次函数 
Function.interval(function(){}, 10);
Function.createDalayed 创建一个延时执行函数 
Function.createDalayed(function(){});
