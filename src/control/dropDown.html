下拉菜单控件

下拉菜单控件是一个抽象的控件基类。控件封装了点击后展开子菜单的逻辑。

基本用法

添加 [data-role="dropDown"]，即可将指定节点初始化为上一个节点的下拉菜单。

我是弹出菜单

为下拉菜单添加 [data-target="{目标节点选择器}"] 可设置目标节点。

我是弹出菜单

API

手动初始化下拉菜单
$('#dropDown1').role('dropDown', {target: '#button1'});
手动显示菜单
$('[data-role=dropDown]').role().show();
手动隐藏菜单
$('[data-role=dropDown]').role().hide();
设置菜单打开后的回调
$('[data-role=dropDown]').role().on('show', function(){   alert("菜单打开了") } );
