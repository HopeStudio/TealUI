弹出层控件

弹出层控件是一个抽象的控件基类。控件封装了鼠标移上后显示弹出层的逻辑。

基本用法

添加 [data-role="popover"]，即可将指定节点初始化为上一个节点的浮层。

我是弹出菜单

为下拉菜单添加 [data-target="{目标节点选择器}"] 可设置目标节点。

我是弹出菜单

API

手动初始化弹出层
$('#popover1').role('popover', {target: '#button1'});
手动显示菜单
$('[data-role=popover]').role().show();
手动隐藏菜单
$('[data-role=popover]').role().hide();
设置菜单打开后的回调
$('[data-role=popover]').role().on('hide', function(){   alert("弹出层关闭了") } );
